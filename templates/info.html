TAP API provides public access to the following capabilities:

Capability Link Summary
Wind data for a point on the map http://dw-tap.nrel.gov/on_map Click on the map to select the desired location. You can also choose the output format.
Wind data for a given address http://dw-tap.nrel.gov/by_address After providing a valid address, you will be able to see wind data for that location. You can also choose the output format.

The capabilities listed below allow to change highlighted parameters in specific requests (read more about parameters):

Capability Example API call Summary
Plot monthly averages https://dw-tap.nrel.gov/monthly?height=65.00m&lat=39.743&lon=-105.147&year=2013 Aggregate to monthly averages and show one or several years of model data.
12x24 (hourly x monthly) summary https://dw-tap.nrel.gov/12x24?height=65.00m&lat=39.743&lon=-105.147 This summary allows seeing diurnal and seasonal patterns in wind data.
Raw time series https://dw-tap.nrel.gov/ts?height=65.00m&lat=39.743&lon=-105.147 Less user-friendly interface that allows external computational tools, dashboards, etc. access TAP's wind data.
Bias-corrected wind speeds(NEW) https://dw-tap.nrel.gov/bc?height=35.00m&lat=41.2&lon=-93.18&year=2011 Apply bias correction using observational data and show monthly average wind speeds, with and without bias correction.
